Damn Vulnerable GraphQL Lab (Node.js)

Welcome to the Node.js reinterpretation of Damn Vulnerable GraphQL Application (DVGA). This environment is designed to teach GraphQL API security through hands-on challenges, demonstrating side-by-side secure coding mitigations in real time.

Quick Start: Choose a module from the grid below or the sidebar. Use the Security Mode toggle in the header to switch the backend between vulnerable and secure resolver paths to inspect how outputs vary.

Vulnerability Map

Architecture Overview

GraphQL Flow

Clients send queries to the unified /graphql endpoint. Standard middleware handles authentication using JSON Web Tokens (JWT). Resolvers query PostgreSQL database using raw pg client queries.

Dual Resolution Paths

Every resolver splits its execution based on the global SECURITY_MODE. This structure allows direct behavioral comparisons between insecure SQL strings and robust parameterized validation layers.
